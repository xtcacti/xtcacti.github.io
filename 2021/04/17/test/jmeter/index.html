what is Jmeter

  • Performance test application
  • Build using java
  • Free & Open Source
  • Recording
  • CLI
  • Reports

install Jmeter

  1. check java is installed on your system
  • java -version on cmd
  1. download Jmeter and unzip
  2. start Jmeter
  • Windows -> jmeter/bin -> jmeter.bat
  • Mac -> open terminal -> jmeter/bin -> sh jmeter.sh

Jmeter Listeners(Reporting)

  • used for reporting
